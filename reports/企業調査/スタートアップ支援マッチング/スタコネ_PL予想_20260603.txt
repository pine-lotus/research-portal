## 5. 主要財務指標

### 5-1. 利益率分析

```
営業利益率：54,551 ÷ 69,725 = **78.2%**
純利益率：37,836 ÷ 69,725 = **54.2%**

【参考：業界水準】
- SaaS企業：50～70%の営業利益率が標準
- VC/アクセレーター：30～50%
- コンサルティング企業：20～40%

→ スタコネの利益率は**異常に高い**（フロー型ビジネスの強み）
```

### 5-2. 人件費効率

```
売上当たり人件費率：9,534 ÷ 69,725 = 13.7%

1名あたりの売上：69,725 ÷ 12 = **5,810万円/人**
1名あたりの純利益：37,836 ÷ 12 = **3,153万円/人**

【解釈】
- 12名で70億円の利益を生む超効率的な組織
- 一般的なコンサル企業：1名あたり3,000～5,000万円の売上
- スタコネは優秀層揃いか、リファラルモデルの効率性を示す
```

### 5-3. 年間キャッシュフロー（概算）

```
営業利益：54,551万円

営業キャッシュフロー ≈ 営業利益 + 減価償却費（小額）
                  ≈ 54,000万円

可処分キャッシュフロー（税引後）：
37,836万円 - 配当・投資等 = 推定 30,000～35,000万円
```

## 6. PL予想の感度分析

現在のPLはかなり楽観的です。以下のシナリオで変動を見てみます。

### シナリオA：出資成功企業が20%に低下した場合

```
スタートアップ出資成功粗利：81 × 20% × 150万 = 2,430万円

総売上：67,855万円
営業利益：54,681万円 → 52,811万円（-1,870万円）
営業利益率：77.8%

影響：比較的小さい（出資成功率による変動）
```

### シナリオB：サポーターチャーンが35%に悪化した場合

```
継続率が65%に低下

サポーター側の新規獲得が必要。
営業強化により、CAC 100万円で新規獲得できた場合：

- 新規獲得コスト：100万 × 30名 = 3,000万円（追加費用）

営業利益：54,551万 - 3,000万 = 51,551万円
営業利益率：73.8%

影響：中程度（スケール時に営業投資が必要）
```

### シナリオC：スタートアップ掲載企業が150社に拡大した場合

```
掲載企業150社、出資成功率35%（現状と同等）
出資成功企業：52社

スタートアップ側売上：
- 掲載年会費：150 × 300万 = 45,000万円
- 出資成功粗利：52 × 150万 = 7,800万円
- 計：52,800万円

営業費用（営業3名追加 × 400万）：
15,174 + 1,200万 = 16,374万円

総売上：41,225 + 52,800 = 94,025万円
営業利益：94,025 - 16,374 = 77,651万円
営業利益率：82.6%

影響：大幅改善（スタートアップスケールの効果）
```

## 7. 現状PLの評価

### ✅ 強み

```
1. 異常に高い利益率（78.2%の営業利益率）
   - リファラルモデルのコスト効率性
   - 両面課金型による多角的な収益源
   - 少人数での高い売上生産性

2. 高いキャッシュフロー（年間3～4億円）
   - シリーズB調達可能性が高い
   - 営業強化への投資余力がある

3. スケーラビリティ（出資成功率35%で掲載150社への成長可能）
   - 営業投資を増やすことで線形成長が可能
   - テクノロジー企業としての典型的な成長曲線
```

### ⚠️ リスク

```
1. 出資成功企業数の推定が不確実
   - 年間28社（35%）というのは仮定
   - 実際には20～40%の変動がありえる
   - ±2,000～2,500万円の利益変動

2. サポーターのチャーン依存
   - チャーン27%は「出資案件がある」前提
   - 出資案件が減少 → チャーン急増のリスク
   - 結果として新規獲得営業コスト増 → 利益圧迫

3. ソルさん依存による売上の脆弱性
   - 出資案件 ≈ ソルさんの投資判断
   - ソルが関心を失ったら → 出資案件が減少
   - 営業利益が78% → 50%台への低下も可能性
```

## 8. 実現可能性の判定

### 現状PL（年間37,836万円の純利益）は実現可能か？

```
判定：⭐⭐⭐⭐ ほぼ現実的

理由：
1. 売上69,725万円は現在の会員数から直線計算
   - 現在のサポーター300名 × 平均150万 = 45,000万
   - 現在のスタートアップ81社 × 掲載300万 + 出資成功 = 24,500万

2. 営業費用15,174万円は業界水準よりむしろ高め
   - イベント開催2,200万円は公開データ（月8～10回）から推定
   - システム開発1,500万円は最近のAI開発投資が見え始めているため妥当

3. 人件費9,534万円は出資情報から矛盾なし
   - シードフェーズで12名の体制は典型的

✓ 現在の77,700万円の営業利益は実現可能
```
