スタコネ_エンジェル税制×シード推計

調査日：2026年6月3日

================================================================================
【推計分析】スタコネの掲載企業における
「エンジェル税制取得 × シード/プレシード企業」の数
作成日：2026年6月3日
================================================================================

## 1. 利用可能なデータ（面談メモから）

【掲載企業の基本情報】
- 掲載企業数：81社
- DD通過率：15%（100社候補中15社のみ）
- 出資総額：20億円（累計）
- 平均調達額：20億 ÷ 81社 = 約2,468万円/社
- マッチング件数：1,701件（累計）

【選定基準（面談で明記）】
「エンジェル税制が通ってるならOK。BPOの会社とかは通過していない、代替可能性が高い」

【重要な示唆】
- エンジェル税制取得は、DD通過の「必須条件」に近い
- つまり、掲載企業のほとんどがエンジェル税制を取得している可能性が高い

================================================================================

## 2. 推計①：エンジェル税制取得企業の割合

### 2-1. 根拠となる事実

```
【スタコネのDD基準】
面談メモより：
「一次産業xAI 、ただのAI銘柄と区別している。
エンジェル税制が通ってるならOK。
BPOの会社とかは通過していない、代替可能性が高い」

解釈：
- 経営革新性：「一次産業×AI」など新しい組み合わせ
- エンジェル税制適格性：個人投資家が投資可能な新規事業
- 代替可能性の低さ：ビジネスモデルが固有性を持つ

→ DD通過企業 ≈ エンジェル税制適格企業 と見なせる
```

### 2-2. 推定値

```
【仮説】
掲載企業81社のうち、エンジェル税制を取得している企業の割合は？

推定方法①：「DD通過率の逆算」
- DD通過率15%というのは、100社候補中15社が通る
- つまり、掲載済み81社は既にDD通過企業の集合
- 掲載企業のほぼ全社がDD基準をクリア済み
- → エンジェル税制取得企業：85～95%

推定方法②：「エンジェル税制の対象性」
- エンジェル税制は「経営革新や事業転換に取り組む企業」が対象
- スタートアップの大多数は該当する
- ただし、形式的な書類申請が必要
- → エンジェル税制取得企業：75~90%

推定方法③：「スタコネの会員構成」
面談メモ：「サポーター側のほぼ経営者。エンジェル税制A活用で個人49名から1.01億円」
- サポーター側：個人125名のほぼ全員がエンジェル投資家
- スタートアップ側も「エンジェル投資対象」でなければ意味がない
- → エンジェル税制取得企業：80~95%

【最終推定】
エンジェル税制取得企業：掲載81社のうち 65～77社
中央値推定：約70社（86%）
```

---

## 3. 推計②：シード/プレシード企業の割合

### 3-1. 調達ステージの定義と平均調達額

```
【一般的な定義】
プレシード：$50k～$500k（500万～5,000万円）
シード：$500k～$2M（5,000万～2億円）
シリーズA：$2M～$15M（2億～15億円）
シリーズB以上：$15M+（15億円以上）

【スタコネの掲載企業の特性】
- 掲載企業：「未調達 or 最初のラウンド」が中心と推定
- 平均調達額：2,468万円
- これは「シード前期」～「プレシード後期」の範囲

【根拠】
1. スタコネの掲載企業は「投資家とマッチング中」
   → つまり「調達が決まっていない」
   → まだシード前のステージが多い

2. マッチング→調達までの期間が6～12ヶ月
   → 掲載企業が「プレシード～シード初期」

3. 平均調達額2,468万円から推定：
   - 2,500万円は、シード初期の典型的な規模
   - 「シリーズA（平均8～10億円）をターゲット」な企業は少ない
```

### 3-2. 掲載企業の調達ステージ分布の推定

```
【分布推定】

出資総額20億円、掲載企業81社から逆算：

プレシード（500万～1,500万円）：
- 企業数推定：25～30社
- 理由：初期段階企業の割合が高い

シード（1,500万～3,000万円）：
- 企業数推定：35～40社
- 理由：スタコネ対象の中心層

シリーズA（3,000万～1億円）：
- 企業数推定：12～16社
- 理由：既に複数ラウンド経験

シリーズB以上（1億円以上）：
- 企業数推定：2～5社
- 理由：後期企業は掲載の必要性が低い

合計：81社
```

**根拠の詳細：**

```
【計算方法】
出資総額20億円を、ステージ別に配分：

プレシード平均額：1,000万円 × 27社 = 2.7億円
シード平均額：2,000万円 × 37社 = 7.4億円
シリーズA平均額：5,000万円 × 14社 = 7.0億円
シリーズB平均額：1億円 × 3社 = 3.0億円

合計：2.7 + 7.4 + 7.0 + 3.0 = 20.1億円 ≈ 20億円 ✓

推定分布：
- プレシード：27社（33%）
- シード：37社（46%）
- シリーズA：14社（17%）
- シリーズB+：3社（4%）
```

### 3-3. シード/プレシード企業の割合

```
【計算】
シード以前（プレシード + シード）= 27 + 37 = 64社
全掲載企業：81社

シード以前の割合 = 64 ÷ 81 = 79%
```

## 4. 推計③：エンジェル税制取得 × シード/プレシード企業

### 4-1. 組み合わせ計算

```
【確認した数値】
① エンジェル税制取得企業：70社（86%）
② シード/プレシード企業：64社（79%）

【交差する企業数の計算】

仮説A：エンジェル税制取得とシード/プレシードが独立
推定数 = 81 × 0.86 × 0.79 = 約55社

仮説B：エンジェル税制取得企業の方が、シード以前に集中
（理由：シード前は個人エンジェル投資が主流。シリーズAはVCからの調達が主流）

推定修正係数：1.1（相関性を考慮）
修正推定数 = 55 × 1.1 = 約61社

仮説C：より保守的な推定
エンジェル税制取得企業の95%がシード以前と仮定：
推定数 = 70 × 0.95 = 約67社
```

### 4-2. 信頼区間付き推定

```
【推定範囲】

楽観シナリオ：
- エンジェル税制取得率：90%
- シード以前の割合：80%
- 推定数：81 × 0.90 × 0.80 = 約58社

基本シナリオ（推奨）：
- エンジェル税制取得率：86%
- シード以前の割合：79%
- 推定数：81 × 0.86 × 0.79 = 約55~58社

保守シナリオ：
- エンジェル税制取得率：80%
- シード以前の割合：75%
- 推定数：81 × 0.80 × 0.75 = 約49社

【結論】
推定範囲：49～67社
最も可能性の高い値：55～58社
中央値推定：約56社（掲載企業の69%）
```

## 5. 推計の検証（別角度からの確認）

### 5-1. 出資総額ベースの検証

```
【検証方法】
シード/プレシード企業の平均調達額で逆算

シード/プレシード平均調達額：推定1,400万円
出資総額20億円 ÷ 1,400万 = 約143社分

ただし：
- 掲載企業81社の出資総額は20億円
- 全企業が調達に成功しているわけではない
- 出資成功企業：約24社（年間）× 複数年 = 累計

累計出資成功企業：推定30～40社
そのうちシード以前：30～40社 × 79% = 24～32社

このうちエンジェル税制取得企業：24~32社 × 90% = 21~29社

これは「調達成功企業」の数であり、「掲載企業全体」ではない
```

### 5-2. DD通過率との相互確認

```
【面談メモより】
「100社候補中、DD通過率15%（=15社）」
掲載企業81社 ÷ DD通過率15% = 過去の候補企業約540社

つまり：
過去540社の候補から、現在81社が掲載
= 掲載企業は「厳選された精鋭」

推定：
掲載企業81社のうち、エンジェル税制取得企業の割合は
「DD通過企業」とほぼ同等 = 85～90%

掲載企業81社のうち、シード/プレシード企業の割合は
「投資ステージ」から推定 = 75～80%

これらは矛盾しない
```

## 6. 最終推定値

```
【結論】

Q：スタコネの掲載企業（81社）のうち、
   「エンジェル税制取得 × シード/プレシード」の企業数は？

A：約50～65社（中央値：約56社、掲載企業の69%）

【根拠サマリー】

1. エンジェル税制取得企業：65～77社（80～95%）
   根拠：DD通過基準にエンジェル税制適格性が含まれる

2. シード/プレシード企業：64社（79%）
   根拠：平均調達額2,468万円 + マッチング中という特性

3. 両条件を満たす企業：約56社（69%）
   根拠：81 × 0.86 × 0.79 = 54.8 → 約55～58社
```

## 7. 経営的含意

```
【スタコネのビジネス戦略への示唆】

掲載企業81社のうち、69%（約56社）が
「エンジェル税制対象 × シード段階」

つまり：
- スタコネの中核市場は「シード企業の個人投資家マッチング」
- エンジェル投資市場の特性（小規模、個人判断）に最適化
- シリーズA以上の企業向けは「副次的」

【今後の成長機会】
1. シード企業の掲載数を増やす（現在の得意領域の深掘り）
   → 調達成功率が高い層

2. シリーズA企業への拡大（アップセル機会）
   → 調達額が大きく、コミュニティ価値が高い

3. プレシード企業への拡大（ダウンセル機会）
   → 数が多いが、調達成功率が低い
```
