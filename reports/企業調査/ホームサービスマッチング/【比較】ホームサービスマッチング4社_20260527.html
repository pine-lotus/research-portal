1. ビジネスモデル比較
═══════════════════════════════════════════

■ LiPLUS（リード課金型）
────────────────────────────────────
「紹介1件ごとに固定料金を業者から受け取る」モデル。
成約・不成約に関わらず、問い合わせを業者に送った時点で課金が発生する。

　ユーザー → 問い合わせ → LiPLUSがマッチング → 業者に紹介 → 紹介料収入（固定額）

　特徴：成約しなくても収入になる。業者のリスクはあるが、LiPLUSは集客数に比例して
　　　　安定した収益を得られる。価格.com傘下に入ったことで集客力の強化が期待される。

■ くらしのマーケット（成功報酬型）
────────────────────────────────────
「予約が成立したときだけ、売上の20%を業者から受け取る」モデル。
業者の初期費用・月額費用は完全ゼロ。

　ユーザー → 比較・予約（オンライン決済）→ 業者が作業 → 成約金額の20%を徴収

　特徴：業者の参入障壁が最も低い。不成約なら費用ゼロなので業者が増えやすく、
　　　　規模の拡大が最も速い。デメリットは空振りが多いと業者の不満につながること。

■ ミツモア（垂直統合型）
────────────────────────────────────
「マッチング（集客）× 業務管理SaaS × B2B調達」の3層で業者の業務全体を囲い込むモデル。

　ユーザー → 最大5社から見積もり → 選んで発注 → 業者が作業 → 成約手数料
　　　　　　　　　　　　　　　　　　　　　　　　　　↓
　　　　　　　　　　　　　　　　　　　　　　　 プロワン（SaaS）で業務管理も一元化

　特徴：業者を「集客だけ」でなく「業務管理」まで囲い込むので離脱しにくい。
　　　　SaaSで安定収益（サブスク）も得られ、LTVが他社より高い構造。

■ ユアマイスター（審査制キュレーション型）
────────────────────────────────────
「量より質」。厳格な審査で厳選したパートナーを紹介し、保証制度で安心を担保するモデル。

　ユーザー → 検索・予約 → 審査済みパートナーが作業 → 成約手数料 + 月額プラン

　特徴：パートナー数は1,500社と最も少ないが、その分ブランド信頼が高い。
　　　　「誰でも登録できるが、依頼は怖い」という大手プラットフォームへの不信を
　　　　逆手に取る差別化戦略。

2. テイクレート（手数料率）比較
═══════════════════════════════════════════

  会社                  課金方式          テイクレート          備考
  ──────────────────────────────────────────────────────────────
  LiPLUS                リード課金         固定単価              3,000〜5,000円/件推定
  くらしのマーケット    成功報酬           20%（明示）           1万円未満は一律2,000円
  ミツモア              成功報酬           8〜35%（公開）        カテゴリにより異なる
  ユアマイスター        成功報酬＋月額     10〜20%（推定）       月額プランとのセット
  ──────────────────────────────────────────────────────────────

■ 読み解きポイント

・くらしのマーケットの「20%明示」は業界で最も透明性が高い。業者にとって計算しやすい。

・ミツモアの「8〜35%」は幅が大きい。高単価の専門サービス（税理士等）は低め、
  汎用サービス（清掃等）は高め。フランチャイズの40〜70%と比較して「安い」を訴求。

・LiPLUSは成約ベースではなく「紹介ベース」なので、業者が受注できなかった場合も
  費用が発生する。業者にとってはリスクがあるが、参入ハードルは月額費用がない分低い。

・ユアマイスターは手数料＋月額プランの複合型で、実質的な費用負担はやや高め。
  ただし審査制による受注品質の高さで受け入れられている。

  薄く広く・効率的に稼ぐモデル。

・くらしのマーケットはアクティブな業者から高い収益を得る「集中型」。
  10万社いても稼ぐのは上位の優良業者に偏っていると推測される。

  700万件のデータが蓄積されれば、AIによるマッチング精度向上で成約率が上がり、
  単位あたりの収益も改善するサイクルが描けている。

■ 4社のセンターピンの「タイプ」の違い

  LiPLUS        → 「流量型」：どれだけ集客できるかが勝負
  くらしのマーケット → 「信頼蓄積型」：口コミというストックが参入障壁
  ミツモア        → 「データ×AI型」：件数が多いほど精度が上がる複利構造
  ユアマイスター  → 「品質保証型」：量より厳選で差別化

■ まとめ：センターピンが示すビジネスの進化方向
  LiPLUSとくらしのマーケットは「今の集客力」で戦っている。
  ミツモアは「データ蓄積による将来の優位性」に賭けている。
  ユアマイスターは「大手との差別化」で生き残りを図っている。

7. 競合構造の整理：誰と誰が競合しているか
═══════════════════════════════════════════

■ 直接競合（同じユーザーを奪い合う）

  くらしのマーケット ←→ ミツモア（業者の掛け持ち登録が多く、ユーザーも両方比較）
  くらしのマーケット ←→ ユアマイスター（ハウスクリーニング領域が重複）
  LiPLUS ←→ くらしのマーケット（ハウストラブル・不用品回収）

■ 間接競合（異なるアプローチで同じニーズに応える）

  ミツモア ←→ LiPLUS（ミツモアは比較型、LiPLUSはお任せ型）
  ユアマイスター ←→ くらしのマーケット（質重視 vs 量重視）

■ 棲み分けの実態

  くらしのマーケット　→　「とにかく多くの選択肢から選びたい」ユーザー向け
  ミツモア　　　　　　→　「複数の見積もりを比べて決めたい」ユーザー向け
  ユアマイスター　　　→　「厳選されたプロに安心して頼みたい」ユーザー向け
  LiPLUS　　　　　　　→　「自分で探さず、すぐ解決したい」ユーザー向け

8. 新規事業担当者向けの示唆
═══════════════════════════════════════════

■ ① 課金モデルの選択がビジネスの性格を決める

  リード課金型（LiPLUS）：安定収益・黒字化しやすい・業者リスクあり
  成功報酬型（くらしのマーケット）：業者が集まりやすい・スケールしやすい・ボラタイル
  複合型（ミツモア・ユアマイスター）：LTVが高い・実装は複雑

  → 新サービスの立ち上げ期は「リード課金型」が最も安全。
    スケールフェーズでは「成功報酬型」に移行するケースが多い。

■ ② センターピンは「量」か「質」か、どちらかに振り切ること

  両方を取ろうとすると中途半端になりやすい。
  ・量（ロングテール）を取るならくらしのマーケット/ミツモア型
  ・質（キュレーション）を取るならユアマイスター型

■ ③ 親会社・資本の有無が将来性を左右する

  LiPLUS → カカクコム傘下でトラフィック爆増の可能性
  ミツモア → 大手金融・建設との連携で保険・不動産領域へ
  くらしのマーケット → 独立継続（IPO or 戦略買収が焦点）
  ユアマイスター → 追加資本の確保が次の課題

  → 「誰と組むか（資本・業務提携）」がスタートアップの成長速度を大きく左右する。
    特にトラフィックを持つプラットフォーマー（カカクコム・Yahoo!等）との提携は
    センターピン（集客力）を一気に強化できる。

■ ④ ミツモアが描く「産業OS」戦略は最も射程が長い

  集客→業務管理→決済→保険→金融と、業者の業務フロー全体を囲い込む設計は、
  単純なマッチング事業では到達できない高いLTVと参入障壁を生む。
  ただし実現までに時間と資金が必要で、先行投資の体力が問われる。

■ ⑤ EV/EBITDAの参考値（M&A相場）

  LiPLUS買収時：EV/EBITDA ≈ 9倍（EBITDA約4.8億円 → 43億円で買収）
  → ホームサービスマッチング分野の黒字企業買収相場として参考になる。

■ ⑥ 「誰も黒字化できていない」ことは参入機会でもある

  くらしのマーケット・ミツモア・ユアマイスターはいずれも赤字。
  LiPLUSのように「小さくても黒字」のモデルには一定の市場価値がある。
  ニッチな業種・地域に特化した「小さなLiPLUS型」ビジネスは、
  M&Aの買収対象としても魅力的な存在になりうる。
