# ゾンタック AI導入プロジェクト — ファイル一覧

菅平のプチホテル「ゾンタック」のフロント業務AI化プロジェクトの資料一式。
（企業研究は `reports/` フォルダ。このフォルダは別プロジェクト用）

## 📁 ファイルの内訳

### 🧑 あなたが見る用
- **当日の進行表_セットアップ手順書.md**
  → 現地（ゾンタック）でやることのチェックリスト。当日これを見ながら進める。

### 🤖 現地PCに置く用（Claude Codeが読む）
`Claude用ファイル（現地PCに置く）/` の中の3つを、フロントPCの作業フォルダにコピーする：

- **CLAUDE.md** … プロジェクトの全体ルール。Claude Codeが毎回これを読んで動く。
- **開発ルール.md** … 守るべきセキュリティ・開発の注意（CLAUDE.mdから参照）。
- **宿情報.md** … 宿の基本情報。返信下書きに使う。ヒアリングで【未確定】を埋める。

## 🚀 現地での使い方（ざっくり）

1. フロントPC（Mac）に作業フォルダを作る（例：`zontag-ai`）
2. 上の3ファイル（CLAUDE.md・開発ルール.md・宿情報.md）をその中にコピー
3. Claude Codeを起動 → 自動でCLAUDE.mdを読み、ルールを守って動く
4. ヒアリングしながら、CLAUDE.md と 宿情報.md の【未確定】を埋める
5. 「当日の進行表」に沿って、返信下書きを1個作る

※ 詳しい手順は「当日の進行表_セットアップ手順書.md」を参照。
